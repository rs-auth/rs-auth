# rs-auth

Composable authentication for Rust with Axum and Postgres.

## Workspace

- `auth/` -> `rs-auth`
- `core/` -> `rs-auth-core`
- `pg/` -> `rs-auth-postgres`
- `axum/` -> `rs-auth-axum`
